IClawMINI Series Comparison

Three product tiers to meet different needs: from basic home cloud storage to high-performance local large model inference.

IClawMINI

Pure OpenClaw Scheduler + Home Cloud Storage
  • SoC: Rockchip RK3568
  • CPU: Quad-core Cortex-A55 @ 2.0GHz
  • NPU: 1 TOPS (optional disabled)
  • Memory: 4GB LPDDR4
  • Storage: 32GB eMMC 5.1 + Dual M.2 NVMe
  • Network: Dual Gigabit Ethernet, Wi-Fi 5, BT 5.0
  • Video: HDMI 2.0 (4K@60fps)
  • Power: 4W idle, 12W full load
  • Cooling: Fully passive
  • Size: 12.7cm × 12.7cm × 4.97cm
Target: Home cloud storage + basic automation, no AI requirements

IClawMINI Pro

Local Small Model Inference (≤2B) + Home Cloud Storage
  • SoC: Rockchip RK3576
  • CPU: 4×Cortex-A72 @ 2.2GHz + 4×Cortex-A53 @ 1.8GHz
  • NPU: 6 TOPS @INT8
  • Memory: 8GB LPDDR4X/LPDDR5
  • Storage: 64GB eMMC 5.1 + Dual M.2 NVMe
  • Network: Dual Gigabit Ethernet, Wi-Fi 6, BT 5.2
  • Video: HDMI 2.1 (4K@120fps / 8K@30fps)
  • Power: 5W idle, 15-20W full load
  • Cooling: Passive + intelligent fan
  • Size: 12.7cm × 12.7cm × 4.97cm
Target: Local voice assistant, lightweight AI, moderate budget

IClawMINI Ultra

High-Performance Local Large Model (7B-30B) + Home Cloud Storage
  • SoC: AMD Ryzen AI Max+ 395 / Intel Core Ultra
  • CPU: 16-core 32-thread Zen 5 (AMD) / High-performance hybrid (Intel)
  • NPU: AMD XDNA 2: 50 TOPS / Intel NPU: 10-40 TOPS
  • Memory: 32GB/64GB LPDDR5/DDR5 (Up to 128GB)
  • Storage: Optional eMMC + Dual M.2 NVMe (PCIe 4.0, RAID)
  • Network: Dual 2.5GbE, Wi-Fi 7, BT 5.4
  • Video: HDMI 2.1 / DP 2.0, multi-display
  • Power: 15W idle, 65W full load
  • Cooling: Heat pipe + active fan
  • Size: 15cm × 15cm × 5cm
Target: High-performance local large models, flagship AI experience
