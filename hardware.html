Hardware Specifications

Precision-machined aviation-grade aluminum body, ultra-compact design

Core Hardware (RK3588 Version)

Processor

  • SoC Rockchip RK3588
  • CPU Octa-core (4×Cortex-A76 @2.4GHz + 4×Cortex-A55 @1.8GHz)
  • NPU 6 TOPS @INT8
  • NPU Support INT4/INT8/INT16/FP16 mixed precision
  • GPU Mali-G610 MP4

Memory & Storage

  • Memory 4GB/8GB/16GB LPDDR4X/LPDDR5
  • Recommended 8GB/16GB for AI workloads
  • Storage 64GB eMMC 5.1
  • Expansion Dual M.2 NVMe slots
  • Interface PCIe 3.0 x1/x2 configurable

Video Capabilities

  • Decoding 8K@60fps
  • Encoding 8K@30fps
  • Output HDMI 2.1 (8K@60fps with HDR)
  • Video Analytics Up to 32 channels @1080p

Network Connectivity

  • Wi-Fi Wi-Fi 6 dual-band (optional Wi-Fi 6E)
  • Bluetooth Bluetooth 5.2 + BLE
  • Ethernet Dual 2.5GbE ports
  • Port Features Link aggregation support

Physical Specifications

Dimensions & Weight

  • Size 12.7cm × 12.7cm × 4.97cm
  • Weight ~0.7kg (without drives)
  • Material Aviation-grade aluminum
  • Design Ultra-compact

Power & Consumption

  • Input 12V DC, 5A
  • Adapter 60W+
  • Idle Power ~8W
  • Typical Power ~25W
  • Max Power ~35W (AI/GPU stress)

Thermal System

  • Cooling Passive + smart fan
  • Fan Control PID controlled
  • Chassis Aluminum alloy
  • Operating Temp 0°C ~ 50°C
  • Industrial Option Available, wider temp range

Interface Layout

  • Front Panel 2×USB-C, audio jack
  • Rear Panel Dual 2.5G ports, USB 3.1, HDMI 2.1
  • Expansion GPIO (I2C/SPI/UART/PWM)
  • Power 12V DC input

Interface Details

Front Panel

2× USB-C (USB 3.1 Gen1, 5Gbps)
One supports DP Alt Mode for video output
3.5mm audio combo jack
Power button
Status LEDs

Rear Panel

2× 2.5GbE ports (with aggregation support)
2× USB 3.1 Gen1 Type-A
1× USB 3.1 Gen1 Type-C OTG
HDMI 2.1 (8K@60fps with HDR)
GPIO expansion header (I2C/SPI/UART/PWM)
12V DC power input
